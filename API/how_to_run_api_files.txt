1. after creating main.py
2. run below command in terminal
    uvicorn main:app --reload
    uvicorn uvicorn <filename_without_.py>:<FastAPI_instance_name> --reload
3. open in browser: http://127.0.0.1:8000
4. Automatic API Documentation
    - FastAPI automatically creates docs.
    - open - http://127.0.0.1:8000/docs
    - This gives: Swagger UI
5. to read database use
    - http://127.0.0.1:8000/users # users if you want to read users
